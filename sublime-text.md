## Atalhos úteis CTRL + P

Navegar por classes e métodos de um arquivo: `@nome_`_\`classe_`_`metodo\`

* **Seleção vertical** 
  * Right Mouse Button + Shift / Add to selection: Ctrl / Subtract from selection: Alt
* **Ir para linha n**
  * Ctrl + G
* **Exibir/Ocultar painel lateral**
  * Ctrl + k + b
